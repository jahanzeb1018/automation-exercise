# automation-exercise

# Ejecución del proyecto

## Cómo ejecutar los tests
Desde la carpeta del proyecto, simplemente ejecuta:

pytest -v

Para ejecutar un test en concreto usar formato:

pytest tests/test_tables.py::test_tables_sort_due_desc_and_check_row2 -v



# Instalación rápida:

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
